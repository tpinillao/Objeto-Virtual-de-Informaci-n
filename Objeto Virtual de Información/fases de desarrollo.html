FASES PROCESO PARA DESARROLLO DE SOFTWARE

Para el desarrollo de software se debe cumplir con el ciclo de vida, el cual permite mediante un proceso construir el paso a paso para la finalización del mismo de forma exitosa cumpliendo con las siguientes Fases:

  • Análisis de requerimiento: En esta fase se debe conocer la necesidad de nuestro cliente y todo lo que se requiere para la realización de nuestro sistema de información.
  • Diseño y arquitectura: En esta fase se informa y determina como funcionara nuestro software y lo que se requiere de herramientas tecnológicas y de red para su uso.
  • Programación: En esta fase se minimiza todo el diseño mediante código y su duración depende del lenguaje de programación que se utilice
  • Pruebas: En esta fase se debe validar que el software este funcionado correctamente según lo solicitado por cliente mediante pruebas que realice un tercero al programador.
  • Documentación: En esta fase se debe elaborar un respectivo manual se usuario con el objetivo de informar al cliente de su manejo y posibles correcciones o mantenimiento futuro.
  • Mantenimiento: En esta fase se busca mejoras para el software, corrigiendo errores presentados o ampliar nuevas funciones del mismo.
